Данная папка предназначена для индивидуальных списков с id пользователей, т.е. возможность каждому аккаунту ставить лайки только из своего списка с id пользователями. Актуально при ведении разных аккаунтов с разными ЦА (целевыми аудиториями).  

Для каждого аккаунта нужно создать свой список.

Формат: логин.txt 
Пример: 76543217654.txt 

!!! Данный список служит как информационный. здесь ничего вносить не нужно.!!!

==================================================================================
